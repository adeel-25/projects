Famous Inventions Progressive Web App

This educational Progressive Web App teaches users about important inventions and inventors that changed the world.

Features

  • Educational invention cards
  • Audio narration
  • Installable application
  • Offline support
  • Responsive design
  • Dynamic JSON content

How to Install

  1. Open the app in Chrome.
  2. Click the Install App button.
  3. The app will install on your device.
