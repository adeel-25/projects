Modifying the Dataset

The application content is stored inside data.json.

Each invention contains:

  • title
  • inventor
  • description
  • image
  • audio

Example JSON Object

{
  "title": "Light Bulb",
  "inventor": "Thomas Edison",
  "description": "Information here",
  "image": "images/thomas.png",
  "audio": "audio/thomasEdison.mp3"
}

Adding New Topics

  1. Add a new object inside data.json.
  2. Add the image into the images folder.
  3. Add the audio into the audio folder.
  4. Update service-worker.js if new files are added.
